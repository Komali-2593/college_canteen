

# 🍽️ Canteen2U – College Canteen Management System

🚀 **Live Website:** https://canteen2u.netlify.app/

Canteen2U is a modern web-based **College Canteen Management System** designed to simplify food ordering for students and streamline canteen operations.  
The platform allows users to browse menus, place orders online, and reduce long waiting queues in college canteens.

---

## 📌 Project Overview

In many colleges, students spend valuable break time standing in long queues to order food.  
**Canteen2U** solves this problem by digitizing the ordering process through an easy-to-use web application.

The system enables:
- Online food browsing
- Quick order placement
- Efficient order handling
- Improved canteen management

---

## ✨ Features

### 👨‍🎓 Student/User Features
- View digital food menu
- Add items to cart
- Place food orders online
- User-friendly interface
- Faster ordering experience

### 🧑‍🍳 Canteen/Admin Features
- Manage food items
- View incoming orders
- Update menu details
- Improve service efficiency

## ⚙️ Installation & Setup (Local)

Follow these steps to run the project locally:

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Komali-2593/college_canteen.git
````

### 2️⃣ Move Project Folder

Copy the project folder into:

```
xampp/htdocs/
```

### 3️⃣ Start XAMPP

Start:

* Apache
* MySQL

### 4️⃣ Create Database

* Open **phpMyAdmin**
* Create a database (example: `canteen_db`)
* Import the provided SQL file

### 5️⃣ Run the Project

Open browser:

```
http://localhost/canteenu
```
